## About The Project

<!-- [![Product Name Screen Shot][product-screenshot]](https://example.com) -->

This website was built as a showcase project for my Curriculum Vitae

The goal of the project is to:
* Give a brief introduction of who I am, what technologies I worked with so far in my career and what projects I've delivered
* Display easy access to all my career-related social media, contact information and various links
* Give freedom and showcase my front-end skills

The design and contents will change over time, reflecting my evolving work situation and the growing list of personal applications and projects.

### Built With

* HTML
* CSS
* JavaScript
